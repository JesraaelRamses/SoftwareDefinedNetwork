# SDN
This repository contains the infrastructure-as-code and 
configurations to deploy an automation and monitoring stack 
in a Software-Defined Networking (SDN) environment using 
containers with Podman and Docker.

It includes:
- AWX (Ansible Tower open source) for automation
- Prometheus + Grafana for observability
- Taiga with SNMP and Mailhog for management and testing
- All running on RHEL with Podman

## Technologies
- Podman (rootless containers)
- Docker
- Ansible AWX
- Prometheus, Grafana
- SNMP, Mailhog
- Taiga
- Networking: SDN 
- Dell Router (VEP4600), Switch (E3248P-ON).


# SDN
Este repositorio contiene la infraestructura como código y 
configuraciones para desplegar un stack de automatización con 
monitoreo en un entorno de Redes Definidas por Software (SDN) 
usando contenedores con Podman y Docker.

Incluye:
- AWX (Ansible Tower código abierto) para automatización
- Prometheus + Grafana para observabilidad
- Taiga con SNMP y Mailhog para gestión y pruebas
- Todo corriendo sobre RHEL y Ubuntu con Podman y Docker.

## Tecnologías
- Podman (contenedores rootless)
- Docker
- Ansible AWX
- Prometheus, Grafana
- SNMP, Mailhog
- Taiga
- Redes: SDN
- Dell Router (VEP4600), Switch (E3248P-ON).
